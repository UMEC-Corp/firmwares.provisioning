    2. Provisioning Command Matrix

    5. Recovery and Local Reset

    5. Recovery and Local Reset

    Current `core` recovery is release-driven. The button provides hold feedback first, and destructive reset is committed only on release at or after the configured threshold.

    Current `core` recovery is release-driven. The button provides hold feedback first, and destructive reset is committed only on release at or after the configured threshold.
